#### 找到两个单链表相交的起始节点

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */

public class Solution{
    public ListNode getIntersectionNode(ListNode headA, ListNode headB){
        if(headA == null || headB == null){
            return null
        }
        ListNode A = headA;
        ListNode B = headB;
        while(A != B){
            if(A == null){
                A = headB;
            } else {
                A = A.next;
            }
            if(B == null){
                B = headA;
            } else {
                B = B.next;
            }
        }
        return A;
    }
}
```

中心原理： a + all + b = b + all + a
