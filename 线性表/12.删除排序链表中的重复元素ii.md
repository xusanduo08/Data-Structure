#### 删除排序链表中的重复元素ii

给定一个排序链表，删除所有含有重复数字的节点，只保留原始链表中 *没有重复出现* 的数字。

**示例 1:**

```
输入: 1->2->3->3->4->4->5
输出: 1->2->5
```

**示例 2:**

```
输入: 1->1->1->2->3
输出: 2->3
```

```java
// 解法一：比较难理解，至今还不太理解
class Solution {
  public ListNode deleteDuplicates(ListNode head) {
    if(head == null || head.next == null){
      return head;
    }
    ListNode dummy = new ListNode(0);
    dummy.next = head;
    ListNode prev = dummy;
    ListNode post = head;
    // 前后两个指针，prev和post
    // 如果post.next.val != post.val，考虑一下几种情况，
    // 1.此时如果prev.next==post，则将prev指针向前移动一个；
    // 2.如果此时prev.next != post，则说明post是当前重复节点的最后一个节点
    // 所以要将prev.next移动到post.next位置上，即prev.next=post.next
    // 为了避免后面还有重复的，prev的指针此时不能变
    while(post != null){
      if(post.next == null || (post.next != null && post.val != post.next.val)){
        if(prev.next == post){
          prev = post;
        } else { // 如果两者不相等，说明post是当前重复节点的最后一个节点
          prev.next = post.next;
        }
      }
      post = post.next;
    }
    return dummy.next;
  }
}

// 解法二：递归
class Solution {
  public ListNode deleteDuplicates(ListNode head) {
    if(head == null || head.next == null){
      return head;
    }
    ListNode cur = new ListNode(0);
    cur = head.next;
    // 1 -->1 ---> 1 ---> 2---> 3
    // head cur
    // 1.上面这种情况需要移动cur直到cur.val != head.val
    // 2.并且此时head因为已经是重复节点了，所以不能要了
    // -----else----
    // 1 ----> 2 ----> 3
    // head    cur
    // 3.这种情况递归返回的节点作为head.next节点
    if(head.val == cur.val){
      while(cur != null && head.val == cur.val){
        cur = cur.next;
      }
      head = deleteDuplicates(cur);
    } else {
      head.next = deleteDuplicates(cur);
    }
    return head;
  }
}
```
