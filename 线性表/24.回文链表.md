#### 回文链表

请判断一个链表是否为回文链表。

**示例 1:**

```
输入: 1->2
输出: false
```

**示例 2:**

```
输入: 1->2->2->1
输出: true
```

思路三步走：

1. 快慢指针寻找中点，将原链表一分为二
2. 反转后半段
3. 逐个比较前半段和后半段节点值



```java
public class IsPalindrome {
  public static boolean isPalindrome(ListNode head){
    if(head == null || head.next == null){
      return true;
    }
    ListNode slow = head;
    ListNode fast = head;
    // 1.寻找中点
    while(fast != null && fast.next != null && fast.next.next != null){
      fast = fast.next.next;
      slow = slow.next;
    }

    ListNode last = null;
    last = slow.next; // 如果节点个数为奇数个，则后半段从中点下一个节点开始
    slow.next = null; // 从中间切断原来的链表

    // 2.反转后半段
    ListNode dummy = new ListNode(0);
    ListNode cur = dummy;
    while(last != null){ // 反转后半段
      ListNode next = last.next;
      last.next = cur.next;
      cur.next = last;
      last = next;
    }
    last = dummy.next;
    
    // 3.比较前半段链表和反转后的后半段链表
    while(last != null){
      if(head.val != last.val){
        return false;
      } else {
        head = head.next;
        last = last.next;
      }
    }
    return true;
  }
}
```
