#### 合并两个有序链表

将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

```
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */

class Solution{
    public ListNode mergeTwoList(ListNode l1, ListNode l2){
        ListNode head  = new ListNode(0); // 保存对链表头部的指向
        ListNode result = head;
        
        while(l1 != null && l2 != null){
            if(l1.val < l2.val){
                result.next = l1;
                result = result.next;
                l1 = l1.next;
            } else {
                result.next = l2;
                result = result.next;
                l2 = l2.next
            }
        }
        
        if(l1 == null){
            result.next = l2;
        } else {
            result.next = l1;
        }
        
        return head.next;
    }
}
```
